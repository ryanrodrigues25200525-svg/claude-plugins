# /economic-surprises

## Tools
- openbb-mcp — economy_indicators, FRED series, economy_calendar

```
| Release | Country | Actual | Consensus | Surprise | Direction |
|---------|---------|--------|-----------|----------|-----------|

SURPRISE SCORE BY COUNTRY (positive = beating expectations)
| Country | Surprise Score | Trend | Implication |
|---------|----------------|-------|-------------|
| US | | | |
| EU | | | |
| UK | | | |
| China | | | |
| Japan | | | |

SECTOR IMPLICATIONS FROM SURPRISES
[How data surprises affect different asset classes and sectors]

BIGGEST SURPRISE THIS PERIOD
- [Release]: Expected X, Actual X — Surprise of X%
- Market reaction: [What moved and by how much]

FORWARD EXPECTATIONS
[Are expectations set too high or too low for upcoming releases?]
```
